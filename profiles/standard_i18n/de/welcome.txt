<div id="first-time"><h1 class="title">Willkommen auf Ihrer neuen Drupal-Website!</h1>
Wenn Sie neu sind bei Drupal folgen Sie diesen Schritten um Ihre Website in Minuten einzurichten:

<strong>Schritt 1. Prüfen von Voraussetzungen</strong>

Prüfen Sie die Berechtigungen der Ordner und Dateien die mit <q>Drupal 6 ipkg</q> verwendet werden. Zu prüfen sind Besitzer und Gruppe. Diese müssen derart konfiguriert sein das Webapplikationen wie Drupal Schreibberechtigung haben. Ohne ausreichende Schreibberechtigungen können Sie selbst, beispielsweise per FTP, wie auch Drupal keine Änderungen installierter Themen und Module durchführen.

Zu prüfende Ordner:

sites/all/modules/contrib
sites/all/modules/own
sites/all/themes

Einfacher Test:

Vergleichen Sie Besitzer und Gruppe der Datei <q>install.php</q> mit den genannten Ordnern und den enthaltenen Dateien.

<strong>Schritt 2. Beginnen mit der Verwaltung</strong>

Besuchen Sie den <a href="[[admin]]">Verwaltungssbereich</a> für einen aufgabenbasierten oder <a href="[[admin/by-module]]">modulbasierten</a> Überblick der verfügbaren Optionen zum konfigurieren Ihrer Website.

<strong>Schritt 3. Aktivieren von Funktionen</strong>

Aktivieren Sie Funktionalitäten für Ihre Website - Blogs, Foren, Umfragen, Tags, Kommentare, Bewertungen und mehr. Drupal kommt mit <a href="[[admin/build/modules]]">vielen Modulen</a> zur Verwendung von Sozial-Publishing-Funktionen auf Ihrer Website.

Hunderte von zusätzlichen Drupal 6.x kompatiblen Modulen sind auf drupal.org im <a href="http://drupal.org/project/Modules" target="_blank">Drupal-Module Downloadbereich</a> verfügbar.

<strong>Schritt 4. Mit der Veröffentlichung von Inhalten beginnen</strong>

<a href="[[node/add]]">Erstellen Sie Inhalte</a> für Ihre Website. Je nachdem welche Module Sie aktiviert haben sehen Sie Blogs, Foren, Umfragen und mehr als verfügbare Inhalts-Typen.

<p>&nbsp;</p>

<hr />

<p>&nbsp;</p>

Einige optionale Konfigurationsaufgaben werden unten aufgeführt. Weitere Informationen finden Sie im integrierten <a href="[[admin/help]]">Hilfe-Bereich</a> oder in den <a href="http://drupal.org/handbooks" target="_blank">Online-Drupal-Handbüchern</a>.

<strong>Wählen Sie ein anderes Design</strong>

Beginnen Sie damit indem Sie vom Standart-Theme die <a href="[[admin/build/themes/settings/garland]]">Farben ändern</a>. Um das Design Ihrer Website grundlegend zu aendern besuchen Sie den <a href="[[admin/build/themes]]">Theme-Bereich</a> und wählen eines der verfügbaren Themen als Standard.

Viele zusätzliche Drupal 6.x kompatible Themen sind auf drupal.org im Bereich <a href="http://drupal.org/project/Themes" target="_blank">Drupal-Themen</a> zum Herunterladen verfügbar.

<strong>Konfigurieren Sie die Startseite</strong>

Die Standard-Startseite ist eine Auflistung aller Inhalte die mit der Veröffentlichungseinstellung <q>Auf der Startseite</q> markiert sind. Sie können Ihre Inhalte auf der Seite <a href="[[admin/content/node]]">Inhaltsverwaltung</a> verwalten. Sie können beispielsweise auch einen bestimmten von Ihnen erstellten Inhalt als Startseite festlegen. Sehen Sie sich die Seite <a href="[[admin/settings/site-information]]">Website-Informationen</a> an um die URL der Startseite zu konfigurieren. Sie können dort auch ein Motto und eine Beschreibung Ihrer Website hinzufügen.
